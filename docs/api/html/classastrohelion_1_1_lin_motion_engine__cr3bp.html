Construct a trajectory from linear approximations of the CR3BP EOMs.

This function sets the out-of-plane motion to have zero amplitude

Parameters
Lthe Lagrange point number [1,5]
r0a three-element vector that specifies the initial position of the arc relative to the chosen Lagrange point
typethe type of linearized motion desired
pArcpointer to an arcset to store the linear trajectory in
numNodesthe number of nodes to place around the linear trajectory; this is nodes for the entire propagation (period * num revs)
Returns
a trajectory object containing one revolution of the trajectory. Because this motion is generated from simplified dynamics, no information about the STM or Jacobi Constant is computed. Accelerations are also not computed. These values are all stored as NAN
