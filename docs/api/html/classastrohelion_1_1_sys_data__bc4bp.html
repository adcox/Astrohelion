double astrohelion::SysData_bc4bp::getEpoch0 ( ) const

Retrieve the epoch associated with T = 0 (seconds, J2000, UTC)

The angles Theta0 and Phi0 coincide with time T = 0, but this is NOT the J2000 epoch T = 0; Rather, T = 0 is the epoch relative to some reference epoch T0, which is returned by this function.

double astrohelion::SysData_bc4bp::REF_EPOCH = 172650161
static

Time when geometry is at reference orientation (theta = phi = 0), seconds, J2000, UTC

Initialize static variable: Alignment epoch of the Sun-Earth-Moon System in ephemeris time (seconds past Jan 1, 2000 00:00:00.00 UTC)

Sun lies in the YZ (near the inertial y-axis) plane at 2005 JUN 21 08:31:35 Sun-Earth-Moon alignment (near the inertial y-axis) occurs at 2005 JUN 22 04:11:35 Average of these two times is 2005 JUN 21 18:21:36
