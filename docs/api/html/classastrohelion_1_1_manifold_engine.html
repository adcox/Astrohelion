Compute manifold arcs from a number of points spaced equally around a low-thrust periodic orbit.

Make sure that the STMs along the orbit represent the cumulative evolution rather than the segment-wise evolution. This characteristic can be guaranteed by calling setSTMs_sequence() on the Arcset in question.

Parameters
manifoldTypeThe type of manifolds to generate
pPerOrbitA periodic, CR3BP orbit. No checks are made to ensure periodicity, so this function also performs well for nearly periodic segments of quasi-periodic arcs. If an arc that is not approximately periodic is input, the behavior may be... strange.
numMansThe number of manifolds to generate
tofTime-of-flight along each manifold arc after stepping off the specified orbit. If tof is set to zero, none of the trajectories will be integrated; each will contain a single node with the initial condition.
stepTypeDescribes the method leveraged to step off of the fixed point along the stable or unstable eigenvector
Returns
a vector of trajectory objects, one for each manifold arc.
Exceptions
Exceptionif the eigenvalues cannot be computed, or if only one stable or unstable eigenvalue is computed (likely because of an impropper monodromy matrix)
See also

std::vector< cdouble > *  eigVals_final = nullptr

Compute the eigenvector/eigenvalue pair(s) that matches the desired manifold stability from a CR3BP periodic orbit.

If the stable/unstable subpsace has more than one set of manifolds, the algorithm selects the eigenvalue/eigenvector that is most unstable/stable.

Parameters
manifoldTypeThe type of manifold desired
pPerOrbitpointer to a periodic CR3BP orbit. No checks are performed to ensure periodicity; If the trajectory is not periodic or nearly periodic, you may get unexpected results. It is assumed that the final STM in the orbit object represents the monodromy matrix.
eigVals_finalpointer to a vector of doubles; the computed eigenvalue(s) will be stored here
Returns
The eigenvector(s) (column) that best matches the criteria
Exceptions
Exceptionif the eigenvalue computation is unsuccessful
Excpetionif no stable or unstable eigenvalues are identified
