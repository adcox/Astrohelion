Detailed Description

A data object containing information about an event that may occur during simulation.

Adding a New Event

  • Add a new enumerated type value with detailed documentation of how it will be parsed.
  • Update the initEvent() function
  • Update the getConTypeStr() function
  • Define or identify a constraint type that can locate this event to a high degree of accuracy.
  • Update the getDist() function
  • Add event to list of allowed constraints in relevant DynamicalModel derived classes
