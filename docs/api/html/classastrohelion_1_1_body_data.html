Detailed Description

The body data object provides a way to store and retrieve information about different celestial bodies.

This may be supplemented/usurped by calls to SPICE in the future.

Author
Andrew Cox
Version
May 15, 2015

Constructor & Destructor Documentation

astrohelion::BodyData::BodyData ( std::string  n)

Construct a BodyData object from preloaded data.

Data about a body is loaded from a map that pulls data from the body_data.xml file. This constructor calls SPICE (not thread safe) to determine the ID associated with the body name

Parameters
nname of the body or dynamical object (e.g., a system barycenter).
Exceptions
Exceptionif the body with the specified name does not have a SPICE ID or if the body has not been loaded
